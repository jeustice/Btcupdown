# 🚀 BTCUpDown

A browser game where you predict whether Bitcoin goes **up or down**.

👉 Live: https://btcupdown.pages.dev/

---

## 🎮 Concept

- You get a BTC price snapshot
- You choose:
  - 📈 Up
  - 📉 Down
- Wait a few seconds…
- See if you were right

Simple. Addictive. Brutal.

---

## 🧠 Core Features

- 🔥 Streak system
- 🏆 Rating progression (Bronze → Diamond)
- 🎯 Daily bonuses
- ⚡ Real-time BTC data (API-based)
- 💾 Local storage (progress saved per device)

---

## 🎯 Goal

Test your intuition against the market.
How far can your streak go?

---

## 🛠 Tech Stack

- HTML / CSS / JavaScript
- Public crypto API
- LocalStorage (no backend)
- Deployed via Cloudflare Pages
